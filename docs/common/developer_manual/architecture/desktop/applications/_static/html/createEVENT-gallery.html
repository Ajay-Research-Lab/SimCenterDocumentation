    bknd-0012
        

          StochasticWindWittigSinha
        

        

          Application file creating EVENT.json files for stochastic wind fields based on the Witting-Sinha method. Supports Wind hazard events.
        

      

      

        

          
usage:
  StochasticWind  options

where options are:
  -?, -h, --help                           display usage information
  --filenameEVENT <Event file location>    Location where generated time
                                           history should be stored
  --filenameBIM <BIM file location>        Location where building
                                           information model is stored
  --seed <Seed value>                      Seed value that should be used to
                                           generate time histories
  -r, --getRV                              Flag indicating whether the
                                           generated event file should
                                           specify random variable
